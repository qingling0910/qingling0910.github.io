docker常用命令

作者： qingling0910 时间： March 27, 2022 分类：

1.查看本地镜像

docker images

2.查看 docker 日志

docker logs -f 镜像名

3.使用 docker 启动服务

docker start 镜像名

4.使用 docker 停止服务

 docker stop 镜像名

5.使用 docker 重启服务

docker restart 镜像名

6.删除镜像

docker rm 镜像名/容器ID

7.批量删除镜像

docker ps -a | awk '{print $1}'|xargs sudo docker rm

说明：
此命令前半部分列出所有镜像，{print $1}为打印出镜像的ID

8.显示全部容器

docker ps -a

9.显示当前运行的容器

docker ps

10.删除所有镜像

docker rmi $(docker images | grep -v RESPOSITORY | awk '{print $3}')

11.构建容器

docker build -t 镜像名称 .   #后面的 . 指的是当前文件夹（其实是 Dockerfile 存放的文件夹）

12.建立映像文件 -rm 选项是告诉 docker ，在构建完成后删除临时的 Container， Dockerfile 的每一行指令都会创建一个临时的 Container ，一般这些临时生成的 Container 是不需要的

docker build --rm=true -t loen/lamp.

13.删除所有容器

docker rm $(docker ps -a)
