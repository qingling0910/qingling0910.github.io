mysql count函数

作者： qingling0910 时间： April 11, 2022 分类：

mysql count函数

1.统计所有记录的数量

select count（*) from table_name;

2.统计某列的数量

select count(column_name) from table_name where 条件;

3.统计某列不重复的数量

select count(distinct column_name) from table_name where 条件;

4.统计某列中的值重复了多少次

select column_name count(1) as 结果的列名 from table_name group by column_name;
