安装docker

作者： qingling0910 时间： March 27, 2022 分类：

1.查看服务器内核版本

uname -r

2.使用 bing 搜索引擎搜索 docker 官网
更新apt软件包索引，并安装最新版本的Docker Engine和containerd，或转至下一步安装特定版本

  • sudo apt-get update
  • sudo apt-get install docker-ce docker-ce-cli containerd.io

3.查看是否安装成功
docker images
