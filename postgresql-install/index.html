postgresql安装

postgreSQL

在服务器上安装postgresql

1、查看Linux类型 cat /etc/os-release ￼

2、使用dnf包管理器安装postgresql dnf install postgresql-server ￼

3、安装完毕后，系统会创建一个数据库超级用户 postgres，密码为空。 sudo -i -u postgres

4、使用命令进入Postgres，输出以下信息，说明安装成功 psql

￼

5、退出postgres \q

6、postgresql安装成功后，默认为开启状态，开启命令 sudo /etc/init.d/postgresql start

7、关闭 sudo /etc/init.d/postgresql stop

8、重启 sudo /etc/init.d/postgresql restart
