索引

一、有哪些索引
mysql一共有5种索引，如下所示：

唯一索引(UNIQUE)
主键索引(PRIMARY)
普通索引(NORMAL)
联合索引()
全文索引(FULLTEXT)

二、索引命名规则
主键索引名为pk_字段名；【强制】 —-pk 即primary key
唯一索引为uk_字段名； —- unique key
普通索引为idx_字段名 —- index的简称

三、创建索引
1、普通索引
1）建立索引
CREATE INDEX 索引名 ON 表名(字段名); 或
ALTER TABLE 表名 ADD INDEX 索引名(字段名);

2、唯一索引
1）建立索引
CREATE UNIQUE INDEX 索引名 ON 表名(字段名); 或
ALTER TABLE 表名 ADD UNIQUE INDEX 索引名(字段名);

3、主键索引
注意事项：
字段不能为null，不能重复；
表只能创建一个主键索引

1）创建索引
ALTER TABLE 表名 ADD PRIMARY KEY(字段名);

4、联合索引
1）创建索引
CREATE INDEX 索引名 ON 表名(字段名1,字段名2);

四、删除索引
DROP INDEX 索引名 ON 表名;

五、修改索引
ALTER TABLE 表名 RENAME INDEX 旧索引名 TO 新索引名;

六、查询索引
1）查询表中所有索引
show index from 表名;
show keys from 表名;

左前原则

①创建组合索引

CREATE INDEX 索引名 ON 表名(字段名1,字段名2); 1 例子： CREATE INDEX cbt_col ON 表名(字段名1,字段名2,字段3); // 根据左前原则能生效的有3个(字段名1,字段名2,字段3)、(字段名1,字段名2)、(字段名1) 如查询语句（生效）： select * from 表名 where 字段名1 = ‘a’ and 字段名2 = ‘b’; // 首先索引出
