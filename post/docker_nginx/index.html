使用docker启动mysql

作者： qingling0910 时间： March 27, 2022 分类：
  1. 拉取 nginx 镜像

docker pull nginx

2.启动 nginx

sudo docker run -d -p 8081:80 –name nginx nginx

  1. 使用 Linux 命令登陆 nginx curl http://121.5.240.206:8081
    说明：
    http://121.5.244.206 为服务器地址
    8081为镜像映射给宿主机的端口
